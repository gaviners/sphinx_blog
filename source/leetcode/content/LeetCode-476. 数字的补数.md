# LeetCode: 476. 数字的补数

[TOC]



## 1、题目描述



给定一个正整数，输出它的补数。补数是对该数的二进制表示取反。

**注意:**

1. 给定的整数保证在32位带符号整数的范围内。
2. 你可以假定二进制数不包含前导零位。

**示例 1:**

```
输入: 5
输出: 2
解释: 5的二进制表示为101（没有前导零位），其补数为010。所以你需要输出2。
```

**示例 2:**

```
输入: 1
输出: 0
解释: 1的二进制表示为1（没有前导零位），其补数为0。所以你需要输出0。
```



## 2、解题思路

​	根据提议分析，首先是正整数，我们只需判断二进制表示中，左面第一个1的位置即可



```c
    if (num == 0) {
        return 1;
    }

    int result = 0;
    int pos;
    for (int i = 31; i >= 0; i--) {
        if ((num & (1 << i) )!= 0) {
            pos = i;
            break;
        }
    }


    for (int i = 0; i <= pos; i++) {
        if ((num & (1 << i) )== 0) {
            result += pow(2, i);
        }
    }

    return result;
```



​	上面的运行效率有点低，实际上，如果考虑异或的话，

```c
int findComplement(int num) {
    
    int temp = num >> 1;
    int xor = 1;
    while (temp > 0) {
        temp >>= 1;
        xor <<= 1;
        xor |= 1;

    }
    return num ^ xor;
    
}
```
